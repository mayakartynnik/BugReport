# bugs.csv
В процессе тестирования мобильного приложения TO DO MVP, инструмента для организации задач, а также функционала корзины и оплаты на веб-сайте интернет-магазина http://demowebshop.tricentis.com/, мы обнаружили некоторые недостатки. Все выявленные баги были зарегистрированы в системе отслеживания ошибок YouTrack. Детальный список всех обнаруженных недостатков можно найти в файле bugs.csv.
